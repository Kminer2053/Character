# 캐릭터 홍보 웹페이지

캐릭터 디자인을 홍보하기 위한 반응형 웹페이지입니다. PC와 모바일 환경에서 모두 최적화되어 있습니다.

## 구조

```
character/
├── index.html          # 메인 HTML 파일
├── styles.css          # 스타일시트
├── script.js           # JavaScript 인터랙션
├── README.md           # 프로젝트 설명서
└── assets/             # 이미지 및 리소스 폴더
    ├── images/         # 캐릭터 이미지
    ├── icons/          # 아이콘
    └── fonts/          # 폰트 (필요시)
```

## 기능

### 섹션 구성

1. **히어로 섹션** - 캐릭터 첫인상 및 메인 비주얼
2. **캐릭터 소개** - 프로필, 성격, 스토리
3. **세계관 & 스토리** - 캐릭터 배경 및 세계관
4. **캐릭터 라이브러리** - 표정/동작/상황별 갤러리 (필터 기능)
5. **굿즈 & 콜라보** - 굿즈 소개 및 콜라보 내역
6. **사용 가이드 & 다운로드** - 이용약관 및 다운로드 링크
7. **뉴스 & 이벤트** - 최신 소식
8. **문의** - 연락처 및 문의 폼

### 주요 기능

- ✅ 반응형 디자인 (PC / 태블릿 / 모바일)
- ✅ 햄버거 메뉴 (모바일)
- ✅ 부드러운 스크롤 앵커
- ✅ 갤러리 필터링 기능
- ✅ 스크롤 애니메이션
- ✅ 위로가기 버튼
- ✅ 문의 폼 (기본 구조)

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 각 섹션의 플레이스홀더 텍스트를 실제 내용으로 교체
3. `assets/images/` 폴더에 캐릭터 이미지 추가
4. HTML의 이미지 경로를 실제 이미지 경로로 수정

## 커스터마이징

### 컬러 변경

`styles.css` 파일의 `:root` 변수에서 컬러를 변경할 수 있습니다:

```css
:root {
    --primary-color: #4A90E2;      /* 메인 컬러 */
    --secondary-color: #F5A623;    /* 보조 컬러 */
    --bg-color: #FFFFFF;           /* 배경색 */
    /* ... */
}
```

### 섹션 순서 변경

`index.html`에서 섹션 순서를 변경하거나 섹션을 추가/제거할 수 있습니다.

### 이미지 추가

1. `assets/images/` 폴더에 이미지 파일 추가
2. HTML에서 플레이스홀더를 실제 `<img>` 태그로 교체

예시:
```html
<!-- 변경 전 -->
<div class="character-placeholder">
    <p>캐릭터 이미지</p>
</div>

<!-- 변경 후 -->
<img src="assets/images/character-hero.png" alt="캐릭터 메인 이미지">
```

## 주의사항

- 현재 문의 폼은 클라이언트 사이드에서만 동작합니다. 실제 서버 연동이 필요하면 백엔드 개발이 필요합니다.
- 이미지는 플레이스홀더로 되어 있으므로 실제 이미지로 교체해야 합니다.
- 모든 텍스트는 예시이므로 실제 캐릭터 정보로 수정해야 합니다.
